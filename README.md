# Arduino Motion Detector
## Overview

This project is a motion detection system built with:
- Arduino Uno
- PIR Motion Sensor (HC-SR501)
- LEDs
- Passive Buzzer
- Push Button
The system detects movement using the PIR sensor and activates an alarm with visual and sound indications.
---
## Features

- Motion detection using PIR sensor
- Red LED alarm indicator
- Passive buzzer alarm
- Green LED for standby mode
- Yellow LED for system activation
- Button to enable or disable the system
- Program written in C (Arduino)
---
## Components
- Arduino Uno
- PIR Sensor HC-SR501
- Passive Buzzer
- 3 LEDs
- Push Button
- Breadboard
- Jumper Wires
